SQL Examples
  • Faculty & Staff
    Entire page is SQL driven. Each link will run process in the code-behind page to search our Colleague SQL database for the correct information. Once this information is retrieved, it dynamically creates tables to display the information.
  • Course Descriptions
    Works the same as the Faculty & Staff page, but also has a back end to link to other courses depending on requirements, and actually searches the text for key symbols to make that information a link.
  • Veterans to Careers
    The "Available Job Positions" section pulls information from SQL database to supply the links, as well as the information that follows on each click, then uses AJAX to update the panels and return the information without reloading the page.
  • ASSIST Corporate Training
    This page contains a hidden section that allows employer applicants to submit job listings and save this information to the database. This infromation is then viewed from an administrator side to allow postings to be viewed on a separate page.

Other Examples
  • Online Application
    While the information given by applicants is saved to a database, there is also operations going on throughout the wizard that save the information entered.
